# EduProfile

EduProfile is a multi-tenant school profiling and student status monitoring system.

## Features
- Tenant management
- Student profiling
- Role and permission management
- Plan management
- Support and updates module

## Tech Stack
- Laravel
- MySQL
- Blade / Vite

## Setup
1. Clone the repository
2. Run `composer install`
3. Copy `.env.example` to `.env`
4. Run `php artisan key:generate`
5. Configure the database
6. Run `php artisan migrate`
7. Run `php artisan serve`
